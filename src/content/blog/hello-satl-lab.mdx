### 수식 테스트

옴의 법칙:

$$
V = IR
$$

MOSFET 드레인 전류 (포화 영역):

$$
I_D = \frac{1}{2} \mu_n C_{ox} \frac{W}{L} (V_{GS} - V_{th})^2
$$

### 코드 테스트
```python
import numpy as np
import matplotlib.pyplot as plt

t = np.linspace(0, 1, 1000)
f = 10  # Hz
signal = np.sin(2 * np.pi * f * t)

plt.plot(t, signal)
plt.xlabel("Time (s)")
plt.ylabel("Amplitude")
plt.title("10 Hz Sine Wave")
plt.grid(True)
plt.show()
```
